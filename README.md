# heart-disease-prediction-ml
# ❤️ Heart Disease Prediction using Machine Learning

## 🚀 Overview

This project predicts the likelihood of heart disease using Machine Learning models based on patient medical data.
It helps in early detection and supports better healthcare decision-making.

---

## 🧠 Features

* Predicts heart disease risk (High / Low)
* Displays probability/confidence score
* Interactive web interface using Streamlit
* Feature importance visualization
* Clean and user-friendly UI

---

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Pandas & NumPy
* Streamlit

---

## 📊 Machine Learning Models

* Logistic Regression
* Decision Tree
* Random Forest (Best Performing Model)

---

## ⚙️ Project Structure

```
Heart-Disease-Prediction/
│
├── app/                # Streamlit UI
├── src/                # ML logic & utilities
├── data/               # Dataset
├── models/             # Saved model (ignored in Git)
├── main.py             # Training script
├── requirements.txt    # Dependencies
└── README.md
```

---

## ▶️ How to Run

### 1. Clone the repository

```
git clone https://github.com/your-username/heart-disease-prediction-ml.git
cd heart-disease-prediction-ml
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Train the model

```
python main.py
```

### 4. Run the app

```
streamlit run app/app.py
```
